Site One Page

Ideal para negócios de pequeno ou médio porte o site One Page é um site onde você inclui todas as informações que o visitante precisa de uma forma objetiva e criativa em um única página

Wireframe

Wireframe é uma representação visual da estrutura básica de uma página. Ele dve ser feito de maneira mais imples possível, mostrando apenas o essencial, como um espécie de rascunho, sem cores ou imagens. Clique com o botão saiba + para obter o wireframe desta página.
